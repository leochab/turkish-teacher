# Turkish Curriculum Map

CEFR-aligned curriculum following the TOMER / Yunus Emre Institute progression.
Each lesson file lives in its level folder: `curriculum/A1/01-greetings.md`, etc.

## Notes for Teachers / Contributors

- Each lesson file should follow the structure defined in `.claude/commands/lesson.md`.
- Lessons should be self-contained: include grammar explanation, 5+ examples with full glosses, exercises with answer key, and vocabulary in the vocab bank format.
- Prerequisites listed above are strict — do not reference concepts from later lessons.
- All Turkish examples must show the gloss: *ev-ler-den* = house-PL-ABL.
