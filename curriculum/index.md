Lessons are generated on-the-fly by the `/lesson` command using this index for prerequisite checking and progression tracking. Mark topics with ✓ when completed.

## How This Works

- Lessons are **generated on-the-fly** by the `/lesson` command — there are no pre-written lesson files.
- This index serves as the **prerequisite graph** and **completion tracker**.
- Mark a topic with ✓ after completing the lesson. The `/lesson` command checks this index before teaching a new topic.
- Prerequisites listed above are strict — the `/lesson` command should not introduce concepts whose prerequisites are unmarked.
- All Turkish examples must show the gloss: *ev-ler-den* = house-PL-ABL.
